HTML

  • Hypertext Markup Language
  • Gives the content structure.
  • The head element containes information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Cascading Style Sheets
  • Gives the content style
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: check what branch we are currently on
  • git pull origin: pulls up latest changes
  • git checkout -b branchname: creates a new branch and switches to it
  • git add -A: stage our changes
  • git commit -m "your message": commit changes
  • git pull origin branchname: make sure branch is upto date before pushing up changes
  • git push origin branchname: pushes changes to the remote repository

Javascript

  • JavaScript (JS), to handle logic and allow users to interact with a website.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script
  • A functions name is followed by parentheses.
  • The code to be executed is written inside the curly brackets.
